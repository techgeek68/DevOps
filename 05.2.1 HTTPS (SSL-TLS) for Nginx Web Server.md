# Configuring HTTPS (SSL/TLS) for Nginx Web Server

HTTPS encrypts communication between web browsers and servers using SSL/TLS certificates, protecting data from interception and ensuring authenticity. SSL/TLS encrypts data between the client and server. TLS 1.2+ provides modern security. Session caching reuses SSL handshake data, reducing CPU overhead by 80/90%. HTTP/2 multiplexes multiple requests over one connection. Online Certificate Status Protocol (OCSP) stapling allows Nginx to provide certificate validity status, eliminating client OCSP lookups. Proper cipher selection strikes a balance between security and performance.

**SSL/TLS:**
 - Cryptographic protocols for secure communication (SSL is outdated; TLS is current)

**Certificate:**
 - A digital document that proves your site's identity

**Certificate Authority (CA):**
  - Trusted organization that issues certificates (e.g., Let's Encrypt, DigiCert)

**SSL/TLS Certificate:**
 - **Let's Encrypt:** Free, automated certificates (90 day validity, auto renewal)
 - **Commercial CAs:** Paid certificates with extended validation options
 - **Self signed:** For testing only (browsers show warnings)

**Basic Nginx SSL Configuration**

```Nginx
server {                                                                                    # Begin server block for HTTPS
    # Enable SSL and HTTP/2
    listen 443 ssl http2;                                                                   # Listen on port 443 with SSL/TLS and HTTP/2 protocol enabled
    server_name example.com;                                                                # Domain name this server block responds to
    
    # Configure certificates
    ssl_certificate /etc/nginx/ssl/cert.pem;                                                 # Path to SSL certificate file (public key + full certificate chain)
    ssl_certificate_key /etc/nginx/ssl/key.pem;                                              # Path to SSL private key file (keep this secure)
    
    # Set protocols and ciphers
    ssl_protocols TLSv1.2 TLSv1.3;                                                           # Only allow TLS 1.2 and 1.3 (disables older insecure versions)
    ssl_ciphers HIGH:!aNULL:!MD5:!3DES;                                                      # Use strong cipher suites, exclude weak/null/old ciphers
    ssl_prefer_server_ciphers on;                                                            # Server chooses cipher (not client) to prevent downgrade attacks
    
    # Enable session cache
    ssl_session_cache shared:SSL:10m;                                                         # Shared memory cache for SSL sessions (10MB stores ~40,000 sessions)
    ssl_session_timeout 10m;                                                                  # SSL session remains valid for 10 minutes before requiring new handshake
    ssl_session_tickets off;                                                                  # Disable session tickets to ensure Perfect Forward Secrecy (PFS)
    
    # Configure OCSP stapling
    ssl_stapling on;                                                                           # Enable OCSP stapling (server fetches certificate revocation status)
    ssl_stapling_verify on;                                                                    # Verify OCSP responses from certificate authority
    ssl_trusted_certificate /etc/nginx/ssl/chain.pem;                                          # CA certificate chain for verifying OCSP responses
    resolver 8.8.8.8 8.8.4.4 valid=300s;                                                       # DNS servers for OCSP lookups (Google DNS), cache results for 300 seconds
    resolver_timeout 5s;                                                                       # Timeout for DNS resolution attempts (5 seconds max)
    
    # Add HSTS header
    add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;         # Force HTTPS for 1 year on domain and all subdomains
}

# Redirect HTTP to HTTPS
server {                                                                                        # Begin server block for HTTP redirects
    listen 80;                                                                                  # Listen on port 80 (unencrypted HTTP)
    server_name www.example.com;                                                                # Domain name to redirect
    return 301 https://www.example.com$request_uri;                                             # Permanent redirect (301) to HTTPS with same URI path
}                                                                                               # End server block
```

**Security Headers**
```nginx
server {                                                                                                 # Begin server block for security headers
    add_header X-Frame-Options "SAMEORIGIN" always;                                                        # Prevent clickjacking by only allowing this site to frame itself
    add_header X-XSS-Protection "1; mode=block" always;                                                     # Enable browser XSS filter and block page if attack detected
    add_header X-Content-Type-Options "nosniff" always;                                                      # Prevent MIME-type sniffing, force browser to use declared content-type
    add_header Referrer-Policy "no-referrer-when-downtrade" always;                                           # Send referrer only when protocol security level stays the same (typo: should be "downgrade")
    add_header Content-Security-Policy "default-src 'self' http: https: data: blob: 'unsafe-inline'" always;   # CSP: Allow resources from same origin, HTTP/HTTPS, data URIs, blobs, inline scripts/styles
}                                                                                                              # End server block
```

- Don't have a domain yet?
  - If you're just testing/learning, then Generate self signed certificate.
   - Create the SSL directory
```bash
sudo mkdir -p /etc/nginx/ssl/sslkey
sudo mkdir -p /etc/nginx/ssl/sslcert
```
  - Generate certificate
```bash
sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 \
  -keyout /etc/nginx/ssl/sslkey/key.pem \
  -out /etc/nginx/ssl/sslcert/cert.pem
```
Parameters:
  * -x509: Create self signed certificate
  * -nodes: No password protection on key
  * -days 365: Valid for 365 days
  * -newkey rsa:2048: Generate 2048-bit RSA key
  * -keyout: Private key file path
  * -out: Certificate file path
  * -subj: Certificate subject (CN=localhost for local testing)
    * Example: `-subj "/C=US/ST=California/L=San Francisco/O=MyOrg/CN=localhost"`

- Generate Self-Signed Certificate
 - Create a certificate directory:
```bash
sudo mkdir -p /etc/pki/tls/private
sudo mkdir -p /etc/pki/tls/certs
```
 - Generate a self signed certificate
```bash
sudo openssl req -x509 -nodes -days 365 -newkey rsa:2048 \
    -keyout /etc/pki/tls/private/localhost.key \
    -out /etc/pki/tls/certs/localhost.crt \
    -subj "/C=US/ST=State/L=City/O=Organization/CN=localhost"
```

- Configure Nginx for SSL
 - Create a Nginx configuration file:
```bash
sudo vim /etc/nginx/conf.d/ssl-website.conf
```
```bash
server {
    listen 443 ssl;
    server_name 127.0.0.1;

    root /usr/share/nginx/html;

    ssl_certificate     /etc/pki/tls/certs/localhost.crt;
    ssl_certificate_key /etc/pki/tls/private/localhost.key;
    ssl_protocols       TLSv1.2 TLSv1.3;
    ssl_ciphers         HIGH:!aNULL:!MD5;
    ssl_prefer_server_ciphers on;

    add_header Strict-Transport-Security "max-age=63072000; includeSubDomains; preload" always;
    add_header X-Content-Type-Options nosniff always;
    add_header X-Frame-Options DENY always;

    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    access_log  /var/log/nginx/ssl_access.log;
    error_log   /var/log/nginx/ssl_error.log;
}

# Redirect HTTP to HTTPS
server {
    listen 80;
    server_name 127.0.0.1;
    return 301 https://$host$request_uri;
}
```

            document.getElementById('pageUrl').textContent = connectionInfo.getFullUrl();
        }

        // Populate modal details
        function populateModalDetails() {
            document.getElementById('tlsSupport').textContent = window.isSecureContext ? 'Supported & Active' : 'Not Supported';
            document.getElementById('userAgent').textContent = navigator.userAgent;
            document.getElementById('timestamp').textContent = connectionInfo.getTimestamp();
        }

        // Refresh connection information
        function refreshConnection() {
            updateStatusVisuals();
            populateConnectionInfo();
            showToast();
        }

        // Show details modal
        function showDetails() {
            populateModalDetails();
            document.getElementById('detailsModal').classList.add('active');
        }

        // Close details modal
        function closeDetails() {
            document.getElementById('detailsModal').classList.remove('active');
        }

        // Show toast notification
        function showToast() {
            const toast = document.getElementById('toast');
            toast.classList.add('show');
            setTimeout(() => {
                toast.classList.remove('show');
            }, 2500);
        }

        // Close modal when clicking outside
        window.onclick = function(event) {
            const modal = document.getElementById('detailsModal');
            if (event.target === modal) {
                modal.classList.remove('active');
            }
        };

        // Initialize on page load
        document.addEventListener('DOMContentLoaded', initializeConnection);
    </script>
